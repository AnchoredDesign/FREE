Welcome to Anchored Design's FREE repo.

We've added this page to make it easier to snag all of our free content with one download. This will include any images, themes and code snippets we've added to our repository. To get individual files, you'll need to visit our github repo by clicking on the View on GitHub button to the left of this text.

For the tech savvy, after you have git installed, you can get the FREE branch using the command line below:

$ cd your_repo_root/repo_name
$ git clone git://github.com/AnchoredDesign/FREE.git
